# EcoWear – Sustainable Fashion Recommendation Tool
![Alt text](UI-image.png)
## 📌 Overview
EcoWear is an AI-powered recommendation tool that helps users find **ethical and sustainable alternatives** to clothing items they’re interested in purchasing online.

The tool allows users to:
- Input an image of a clothing item.
- Detect the **item type** and **brand**.
- Check the brand’s **sustainability and ethical practices**.
- If the brand does not meet sustainability criteria, suggest **3 alternative brands** with similar style and price point.

Our goal is to make **sustainable shopping** more accessible and to reduce **fast fashion waste**.

---

## 🌱 Why EcoWear?
The fast fashion industry contributes massively to **textile waste**, **carbon emissions**, and **poor labor conditions**.  
We wanted to create a tool that:
- Educates shoppers on sustainability.
- Promotes ethical and responsible fashion.
- Makes eco-friendly choices **easy** and **affordable**.

---

## 🚀 Features
- **Image-based search**: Upload an image of a clothing item.
- **Brand check**: See if the brand follows ethical & sustainable practices.
- **Smart recommendations**: Get 3 similar items from better brands.
- **Clean UI**: Easy-to-use interface for quick browsing.

---

## 🛠️ Tech Stack
- **Python** (Flask for backend)
- **Pandas** (data handling)
- **HTML/CSS/JS** (frontend)
- **CSV datasets** (product links & images)
- **AI/ML** (image recognition for product type)

## ⚙️ Installation & Setup
1. **Clone the repository**  
   git clone https://github.com/yukesh1112/ecowear.git
   cd ecowear

2. **Install dependencies**  

3. **Run the app**  
   python server.py

4. **Open in browser**  
   Visit http://127.0.0.1:5000

## 📌 Example
**User uploads**: a picture of a purple dress from a fast fashion brand.  
**EcoWear suggests**:  
- 2 similar purple dresses from sustainable brands.  
- Links to purchase them at a similar price point.
